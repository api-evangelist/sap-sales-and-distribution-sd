# SAP Sales and Distribution (SD) API (sap-sales-and-distribution-sd)
APIs for SAP Sales and Distribution module covering sales orders, pricing,  delivery, billing, and customer management processes


**URL:** [Visit APIs.json URL](https://api.sap.com/sd)

## Tags:

 - SAP, Sales, Distribution, ERP, S/4HANA, OData

- **Modified:** 2024
